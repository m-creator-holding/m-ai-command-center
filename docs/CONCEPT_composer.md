# Концепт командной строки чата (Композер) — «Командный центр»

**Дата:** 2026-06-13 · **Статус:** концепт на согласование Владельца

Композер = поле общения с Альфредом + всё, что Владелец может «подвесить» к
поручению. Цель: возможности шире, чем у ChatGPT, но интерфейс остаётся
минималистичным (принцип Владельца: ничего лишнего, простой деловой язык).
Решение — ПРОГРЕССИВНОЕ РАСКРЫТИЕ: по умолчанию 4 элемента, вся мощь — в
аккуратных меню «по запросу».

## Композер по умолчанию (минимум на экране)
`[ + ]  Сообщение Альфреду…            [ 🎤 ]  [ ↑ ]`
- `+` — меню «Добавить» (вложения, инструменты, отделы, приложения).
- поле ввода (текст).
- 🎤 — голосовое сообщение (диктовка/команда голосом, расшифровка Whisper).
- ↑ — отправить.
Над полем — маленькие съёмные теги выбранных режимов (модель, поиск и т.п.).

## 1. Ввод
- Текст.
- Голос (🎤): надиктовать поручение; расшифровка на нашем сервере (Whisper).
- Вложения: документ/фото/таблица/чертёж/скрин — Альфред анализирует (договор,
  смета, фото объекта, скрин ошибки).

## 2. Выбор «мозга» (режим мышления) — как выбор модели у GPT, по-деловому
- «Быстрый ответ» (быстрая модель) — переписка, простые задачи.
- «Глубокий анализ» (сильная модель + рассуждение) — сложные решения, стратегия,
  безопасность.
- Авто (Альфред сам выбирает по сложности). Тег режима виден над полем.

## 3. Режимы работы (как Поиск/Исследование у GPT — расширенные)
- 🌐 Поиск в интернете — рынок, конкуренты, новости, цены, тендеры.
- 🔬 Глубокое исследование — развёрнутый отчёт (due diligence, анализ рынка/
  конкурента/площадки), со сбором источников.
- 📊 Анализ данных — посчитать по данным проекта/таблицам/выгрузке, построить
  отчёт и график.

## 4. Команды ИИ-агентов (наша уникальная мощь) — «Подключить отдел»
Выбор, кого Альфред подключает к поручению:
- Департаменты: Бэкенд, Фронтенд, Дизайн, Качество и безопасность, Аналитика,
  Инфраструктура, Исследования (R&I).
- Доменные команды пода (для НЗСН: Контент, Каталог, SEO, Тендеры, Продажи/SDR).
- Уровень мощности: один помощник · отдел · большая волна (5–15 сотрудников
  параллельно).
- Призыв конкретной роли через @ (как упоминание): «@Аналитик тендеров»,
  «@Дизайн-директор». Альфред оркестрирует выбранных и сводит результат.

## 5. Приложения и инструменты на сервере (наша уникальная мощь) — «Инструменты»
Что уже установлено и Альфред может задействовать:
- 🖼️ Генерация изображений (Nano Banana Pro / Imagen) — обложки, ассеты, текстуры.
- 🧊 Blender — 3D-рендеры (визитки, упаковка, каски, изделия).
- 🌐 Браузер-автоматизация (Playwright) — проверить сайт, заполнить, снять скрин.
- 🗣️ Расшифровка речи (Whisper) — звонки, совещания.
- 📄 Документы — Word/Excel/PDF/презентации (сделать КП, отчёт, договор-черновик).
- 🚀 Публикация сайтов (Cloudflare) — выложить/обновить (за «красной зоной»).
- 📈 Графики и дашборды.
Список — расширяемый: поставили новую программу — появилась в меню.

## 6. Рабочая область
- 📁 Выбор проекта/папки, к которой Альфред получает доступ в этой задаче (по
  умолчанию — текущий проект). Гарантирует: не лезет в чужое.

## 7. Внешние подключения (каждое — за «красной зоной», по разрешению)
1С, банк, Росреестр, почта, Telegram, тендерные площадки. По умолчанию выкл,
включаются осознанно, требуют подтверждения на действие.

## 8. Поручения и автоматизация
- ▶️ Поставить задачу (Альфред декомпозирует и раздаёт отделам).
- 🔁 Повтор/расписание: «каждое утро — сводка по холдингу», «еженедельный дайджест».
- ⭐ Шаблоны быстрых команд: «Сводка по холдингу», «Проверь сайт», «Подготовь КП»,
  «Анализ конкурента».

## 9. Контекст и безопасность (всегда видно, ненавязчиво)
- Контекст-тег: с кем говорю — «Альфред · НЗСН · Сайт» + что подключено (модель,
  режим, отделы, инструменты) маленькими тегами над полем; лишнее — убирается.
- 🛡️ Красная зона: действия с «руками» (правка/публикация/удаление/деньги) —
  только с подтверждением. Тумблер автономности: «спрашивай каждый шаг» /
  «делай и отчитывайся».

## 10. Память — ФУНДАМЕНT (не доп-функция; уточнение Владельца msg 6924)
Память — основа системы, работает АВТОМАТИЧЕСКИ и ВСЕГДА, а не по кнопке:
- Всё хранится на нашем сервере: каждый диалог, действие, изменение по каждому
  проекту/направлению. Ничего не теряется; Альфред помнит всю историю проекта.
- Память постоянно ПОПОЛНЯЕТСЯ: решения, факты, договорённости, предпочтения
  Владельца накапливаются → чем дольше работаем, тем контекстнее Альфред. (Так
  уже работает сейчас: memory-файлы + история сессий — переносим в продукт.)
- 📌 «Запомни» — НЕ сама память, а ручная закладка: подсветить особо важное
  правило/факт, чтобы держалось на видном месте и применялось всегда.
- Всё под контролем Владельца: посмотреть / найти / удалить.

## UX-рекомендация (чтобы мощь не превратилась в перегруз)
- По умолчанию: `+`, поле, 🎤, ↑ — и всё. Чисто, как у GPT.
- `+` открывает ОДНО аккуратное меню с группами: Добавить (вложения) · Режим
  (поиск/исследование/анализ) · Подключить отдел · Инструменты · Приложения ·
  Подключения. Простой деловой язык, иконки.
- Выбранное показывается съёмными тегами над полем — видно, что «подвешено».
- Частое (модель, голос) — сразу в строке; редкое — в меню.
- Принцип: «сложное — спрятано, простое — на виду».

## Что делает это лучше GPT
1. Не просто модель отвечает — за полем стоят РЕАЛЬНЫЕ команды ИИ-сотрудников и
   программы сервера, которые делают работу.
2. Привязка к проекту/папке + изоляция + «красная зона» — безопасный контроль.
3. Расписание и шаблоны — холдинг работает сам, Альфред докладывает.
4. Один композер управляет всем холдингом — от поста в TG до правки сайта НЗСН.
